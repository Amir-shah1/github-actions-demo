🚀 CI/CD Pipeline in Action

This project is automatically deployed using GitHub Actions.

What happens here?

  • I push code to GitHub
  • GitHub Actions automatically runs a workflow
  • The site is built and deployed
  • This page updates live without manual deployment

Why this matters

This is a basic CI/CD (Continuous Integration & Continuous Deployment) pipeline. It ensures that every change is automatically tested and deployed.
